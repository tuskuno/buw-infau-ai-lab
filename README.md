# AI Lab - InfAU Workshop

A hands-on workshop where Bauhaus InfAU staff use AI-assisted coding to build an isovist web app — no prior web development experience needed.

Everyone builds the same baseline: a 3D Weimar city view with isovist visualization using Three.js. Then each participant customizes and extends it their own way.

## What's Inside

| Folder | Contents |
|--------|----------|
| `workshop/` | Workshop guide, starter prompt, cheatsheet, and Weimar GeoJSON data |
| `workshop/setup-guide.md` | Pre-workshop setup instructions (install Git, VS Code, Node.js, Claude Code) |
| `workshop/resources.md` | Curated list of tools, data sources, and learning materials |

## Getting Started

**Before the workshop**, follow the [setup guide](workshop/setup-guide.md):

1. [Git for Windows](https://git-scm.com/downloads/win) (Windows only)
2. [VS Code](https://code.visualstudio.com/download)
3. [Node.js LTS](https://nodejs.org/)
4. [Claude Code](https://code.claude.com/docs/en/getting-started)
5. [Claude Pro or Max](https://claude.ai/pricing) subscription
6. [GitHub account](https://github.com/signup) (free)

**During the workshop**, follow the [workshop guide](workshop/README.md).

## Results

Each participant deploys their own app under the `bauhaus-infau` org at `https://bauhaus-infau.github.io/isovist-<name>/`.
